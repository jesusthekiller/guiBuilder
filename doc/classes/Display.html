Class Display

Display class unifying all types of displays.

Many of theese functions are simple wrappers around Term API. Wrappers (not only functions described as 'wrapper' but also those without description) are passing all arguments to native function via ( ... ), everything is returned as well.

Depends on: > Middleclass
> Connection

Usage:

    Display = dofile("display.lua")

Info:

  • Release: 0.01 (indev)
  • Author: jesusthekiller
