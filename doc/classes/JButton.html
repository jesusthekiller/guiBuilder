* textPressed (Color when button is pressed

Note: if color will be left unspecified, default from JButton._DEFAULT_COLOR will be used.

Parameters:

  • position Position
  • color Color
  • timeout number Click animation length (in seconds). Optional, default = JButton._DEFAULT_TIMEOUT.
